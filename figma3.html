Apprendre le CSS

CSS signifie Cascading Style Sheets. Il a été crée durant les années 90, il permet d'ajouter des styles au HTML. Avec le CSS vous pouvez changer la couleur, la taille et de créer des animations.

h1 {
color : #fff;
font-size: 25px;
font-weight: 700;
}
